## Description
此适配器从网页上读取Epson Stylus Photo PX830的墨盒状况

![旧文本](zh-cn/adapterref/iobroker.epson_stylus_px830/../../../de/adapterref/iobroker.epson_stylus_px830/img/printer_website.png "EPSON Stylus Photo PX830网站")

##设置
### IP /端口
本地网络中打印机的IP地址。没有输入，就没有查询。端口号是可选的，仅在输入IP地址时考虑。

###同步
ioBroker查询打印机之间的持续时间。输入是可选的。默认为180分钟（每3小时）。

##激活
适配器按照设置中的指定按时间顺序启动。
